## Codebase Patterns
- Weight initialization follows truncated LeCun normal: `trunc_normal_init_(tensor, std=1.0 / (in_features ** 0.5))` from `models/common.py`
- Use `nn.Parameter` with `trunc_normal_init_` directly (see `CastedLinear` in `models/layers.py`)
- Forward passes use `F.linear(input, weight, bias)` pattern rather than `nn.Linear` modules
- No formal typecheck config (pyright/mypy) exists; use `python -m py_compile` for basic checks
- HookContext dataclass uses `Optional[torch.Tensor] = None` for fields that should be backward-compatible (existing hooks ignore new fields)
- `input_embeddings` is computed in `_Inner.forward()` at the start and passed to all HookContext sites
- The `collect_residual_streams` hook in `hrm_inspect.py` constructs HookContext with keyword args — new fields must have defaults to avoid breaking it
- In forward(), grad-step L-hook fires before H-level (ctx.z_H = z_H_before), grad-step H-hook fires after H-level (ctx.z_H = z_H_after/z_H*)
- Manual ACT loop pattern: `model.empty_carry(batch_size)` → `model.reset_carry(ones, carry)` → loop `model(carry, batch, act_step=step)`
- Residual streams have shape [batch, seq_len + puzzle_emb_len, hidden]; strip puzzle_emb_len prefix to align with labels [batch, seq_len]
- Blank cells in Sudoku: `inputs == 1`; digit labels are tokens 2-10 → remap to classes 0-8 via `labels - 2`
